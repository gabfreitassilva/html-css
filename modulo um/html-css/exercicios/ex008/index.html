Principais formatações

Negrito / Destaque

Nesta linha existe um termo em negrito usando a tag <b>

Agora nesta linha, temos um termo em destaque usando a tag <strong>

Ítalico / Ênfase

Nesta frase temos um termo em ítalico usando a tag <i>

Agora nesta frase temos um termo em ênfase usando a tag <em>

Marcação de texto

Podemos também criar um texto marcado usando a tag <mark>

E agora temos outro texto marcado, se liga vei

Texto grande e Texto pequeno

Neste paragrafo vamos criar um texto grande com a tag <big> e também um texto pequeno com a tag <small>

Texto deletado

Podemos marcar um pedaço do texto, frase ou afins, como excluído, para que na leitura ele não seja totalmente considerado usando a tag <del>

Texto inserido

Já no texto inserido, podemos marcar uma parte do texto com um sublinhado para informar que ele foi adcionado depois da edçião ou dar um ênfase diferente usando a tag <ins>

Existe também a tag <u> para marcar um texto como sublinhado

Texto sobescrito e Texto subescrito

Um texto sobescrito pode ser potências na mátematica 72 usando a tag <sup>

Assim como também podemos escrever formulas químicas H2O usando a tag <sub>
