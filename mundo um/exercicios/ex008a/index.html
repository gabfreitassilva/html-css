Principais formatações

Negrito / Destaque

Usando tag b para negrito (não semântica)

Usando a tag strong para destaque (semântica)

Itálico / Ênfase

Usando a tag i para itálico (não semântica)

Usando a tag em para ênfase (semântica)

Texto marcado

Usando a tag mark para marcar

Texto grande e pequeno

Usando big para texto grande (obsoleto)

Usando small para texto pequeno

Texto deletado

Usando a tag del para marcar parte do texto que devemos desconsiderar, coisas do tipo

Texto inserido

Usando a tag ins para marcar parte do texto como texto inserido

Existe também a tag u para sublinhar textos (não semântica)

Texto sobrescrito

Usando a tag sob como x20

Texto subscrito

Usando a tag sub como H2O
